# Простой разметчик изображений для YOLO

Файл программы: [yolo_labeler.py](/C:/Users/slywater/Projects/neuro/6_Image_detection/yolo_labeler_app/yolo_labeler.py)

## Что умеет

- открывает папку с изображениями;
- принимает `jpg`, `jpeg`, `jpe`, `jfif`, `png`, `bmp`, `webp`;
- позволяет мышью рисовать прямоугольники;
- сохраняет разметку в формате YOLO (`class_id x_center y_center width height`);
- хранит классы списком, по одному на строку в панели слева;
- добавляет новый класс кнопкой `+`;
- запоминает последнюю папку и список классов в `labeler_config.json`;
- обновляет список изображений кнопкой `Обновить картинки`, если в папку докинули новые файлы;
- переключает изображения стрелками `←` / `→`;
- переключает активный класс клавишами `A` / `D` по порядку;
- переключает активный класс клавишами `0-9`;
- удаляет выбранную рамку клавишей `Delete`;
- отменяет последнюю добавленную рамку клавишей `Ctrl+Z`, в том числе на русской раскладке.

## Запуск

```powershell
cd C:\Users\slywater\Projects\neuro\6_Image_detection\yolo_labeler_app
pip install pillow
python yolo_labeler.py
```

## Как устроено сохранение

- разметка всегда сохраняется внутри выбранной исходной папки с изображениями;
- для выбранной папки `...\images\train` файлы разметки попадут в `...\images\train\labels`;
- если раньше разметка лежала в старом соседнем пути вроде `...\labels\train`, программа сможет её прочитать, но после сохранения запишет уже в новый путь внутри исходной папки.

Также в выбранной исходной папке создаётся файл `classes.txt` со списком классов.

## Рекомендуемая структура датасета

```text
dataset/
  images/
    train/
      labels/
    val/
      labels/
  classes.txt
```

## Примечание

Это минимальный инструмент. Если потом понадобится:

- автоматическое деление на `train/val`;

это можно добавить отдельным следующим шагом.
